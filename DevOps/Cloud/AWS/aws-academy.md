# Educational content AWS

Basically, it is a platform that offers courses and learning resources that enable students to develop different skills related to the AWS cloud. Approved educators have access to all AWS Academy courses and learning labs

## Contents: 
- [AWS Academy Cloud Foundations](https://aws.amazon.com/es/training/awsacademy/)
- [Cloud Academy AWS](https://cloudacademy.com/library/amazon-web-services/)
- [Udemy Stephane Maarek](https://www.udemy.com/user/stephane-maarek/)
- [AWS Academy Cloud Architecture](https://aws.amazon.com/es/training/awsacademy/cloud-computing-architecture/)
- [Cloud Quest AWS](https://aws.amazon.com/es/training/digital/aws-cloud-quest/)
- [FooBar Serverless - Youtube Channel](https://www.youtube.com/channel/UCSLIvjWJwLRQze9Pn4cectQ)
